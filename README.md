# 🎓 大学生压力分析与管理导航站 (Group C38)

这是一个基于 **LIF001** 课程调研成果开发的交互式 Web 应用程序，旨在分析大一新生的压力现状并提供科学的应对方案。

## 🚀 项目功能
- **📊 数据洞察**：可视化展示关于压力来源、缓解方式及睡眠影响的调研数据。
- **🌿 生存指南**：针对学业、社交和心态提供定制化的建议（Product Development Plan）。
- **📢 匿名树洞**：为用户提供一个释放情绪的交互空间。

## 🛠️ 技术栈
- **Python**: 核心编程语言。
- **Streamlit**: 快速构建 Web 应用框架。
- **Plotly**: 生成动态交互式图表。
- **Pandas**: 数据处理与管理。
